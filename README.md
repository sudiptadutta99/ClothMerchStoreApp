# 🛍️ Online B2C E-Commerce Store – Clothes & Merchandise

## 📌 Overview

This project is a fully functional **Business-to-Consumer (B2C) e-commerce platform** built with **ASP.NET MVC** and **SQL Server**.
It allows customers to browse and purchase clothes & merchandise online, manage profiles, maintain wishlists, and place orders.
An **admin panel** is provided for product, category, and order management.

---

## 🚀 Features

### **User Module**

* User Registration & Login (secure authentication)
* Profile & Address Management
* Wishlist with “Move to Cart” feature
* Product Reviews & Ratings

### **Shopping Module**

* Product catalog with category & filter options
* Shopping Cart with quantity updates
* Checkout process with order summary
* Payment integration placeholder

### **Admin Module**

* Product and Category Management
* Order Management (status updates)
* Inventory Management

## ⚙️ Installation & Setup

1. **Clone the Repository**

   ```bash
   git clone https://github.com/yourusername/your-repo-name.git
   ```

2. **Open in Visual Studio**

3. **Set Up Database**

   * Update `appsettings.json` or `Web.config` with your SQL Server connection string.
   * Run migrations or execute SQL script to create the database.

4. **Run the Application**

   * Set the project as the startup project.
   * Press **F5** or click **Run** in Visual Studio.
